# 🎯 MKBotX_Bot (v0.0.1)

**MKBotX** helps you create your own Telegram bots with advanced menus. From Games to Online Stores.

---

### (🌐) Supported Languages
EN, AR, RU (+10 more)

### 👾 (☰) In-Menu Features ⠿
* **Create menu:** Dynamic and static buttons.
* **Mailing:** Broadcast messages to users.
* **Forms:** Feedback and data collection forms.
* **Advanced:** Macros, Referral system, Balance, Auto Payments.

### 🛒 In-Shop Capabilities 💸
* Product Categories & Shopping Cart.
* Client Profiles & Order Management.

### 🎩 Ad Market 🎭
* Send Ads to over 7 Million people.
* Receive Ads to earn rewards.
